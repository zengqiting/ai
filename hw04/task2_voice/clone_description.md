# 任务二：剪映声音克隆说明

## 一、克隆过程

| 项目 | 内容 |
|------|------|
| 使用工具 | 剪映专业版 |
| 工具版本 | 6.0.0 |
| 克隆方式 | 录制个人声音样本 |
| 录音时长 | 约15秒 |
| 脚本来源 | 任务一：大模型生成文稿《当AI学会了我的声音》 |

## 二、操作步骤

1. 打开剪映专业版，创建新项目
2. 点击“文本” → “新建文本”，粘贴任务一的文稿
3. 选中文本素材，点击右侧面板的“朗读”
4. 在弹出的窗口中，点击“克隆音色” → “+”
5. 按提示朗读一段文字，完成声音克隆
6. 选择克隆好的音色，点击“开始朗读”
7. 等待生成完成，点击“导出” → 仅导出音频（MP3格式）

## 三、导出结果

| 项目 | 内容 |
|------|------|
| 文件格式 | MP3 |
| 文件时长 | 约45秒 |
| 文件大小 | 约1.2MB |
| 文件路径 | cloned_audio.mp3 |

## 四、效果说明

克隆后的声音能够较好地还原本人音色特征，包括：
- 音色相似度：约85%
- 语调节奏：基本一致
- 发音清晰度：良好

## 五、注意事项

- 克隆时需要环境安静，录音清晰
- 朗读脚本时建议保持自然语速
- 导出时选择音频格式，便于任务三识别
