## 选题简介

本视频围绕“智能机器人与具身智能”主题，探讨机器人是否会取代人类这一问题。视频从历史视角出发，分析机器人取代的是“重复性工作”而非“创造性工作”，最终得出结论：机器人不会取代人类，但会用机器人的人可能更有竞争力。
